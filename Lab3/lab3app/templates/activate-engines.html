تفعيل محركات القوالب في Django

دليل عملي خطوة بخطوة لتفعيل DTL (الافتراضي)، Jinja2، وMako في مشروع Django.

1. تفعيل DTL (Django Template Language)

DTL مُفعّل افتراضيًا في Django. لا يتطلب تثبيتًا إضافيًا.

1

أضف مجلد templates في مشروعك

your_project/
├── templates/
│   └── index.html
├── settings.py
└── urls.py
2

تأكد من إعدادات القوالب في settings.py

TEMPLATES = [
    {
        'BACKEND': 'django.template.backends.django.DjangoTemplates',
        'DIRS': [BASE_DIR / 'templates'],
        'APP_DIRS': True,
        'OPTIONS': {
            'context_processors': [
                'django.template.context_processors.debug',
                'django.template.context_processors.request',
                'django.contrib.auth.context_processors.auth',
                'django.contrib.messages.context_processors.messages',
            ],
        },
    },
]

2. تفعيل Jinja2

Jinja2 يوفر مرونة أكبر. يتطلب تثبيت حزمة إضافية.

1

تثبيت الحزمة

pip install Jinja2
2

تعديل settings.py لاستخدام Jinja2

TEMPLATES = [
    {
        'BACKEND': 'django.template.backends.jinja2.Jinja2',
        'DIRS': [BASE_DIR / 'jinja2'],
        'APP_DIRS': True,
        'OPTIONS': {
            'environment': 'myproject.jinja2.environment',
        },
    },
    # مهم: اترك DTL للتطبيقات المدمجة مثل admin
    {
        'BACKEND': 'django.template.backends.django.DjangoTemplates',
        'APP_DIRS': True,
        'OPTIONS': {
            'context_processors': [...],
        },
    },
]
3

إنشاء ملف jinja2.py لتهيئة البيئة

# myproject/jinja2.py
def environment(**options):
    from jinja2 import Environment
    
    env = Environment(**options)
    env.globals.update({
        'my_global': 'value',
    })
    return env

3. تفعيل Mako

Mako يوفر أداءً عاليًا، لكنه يتطلب تكامل يدوي.

1

تثبيت الحزم

pip install Mako django-mako-plus
2

تعديل settings.py

TEMPLATES = [
    {
        'BACKEND': 'django_mako_plus.MakoTemplates',
        'DIRS': [BASE_DIR / 'mako_templates'],
        'APP_DIRS': True,
        'OPTIONS': {
            'input_encoding': 'utf-8',
        },
    },
    # DTL للتطبيقات الأخرى مثل admin
    {
        'BACKEND': 'django.template.backends.django.DjangoTemplates',
        'APP_DIRS': True,
    },
]
3

إنشاء قالب Mako (index.html)

<%page args='name'%>

  
    
مرحباً ${name}!

نصائح مهمة

  • يمكن دمج أكثر من محرك في نفس المشروع.
  • تأكد من أن APP_DIRS مضبوط بشكل صحيح.
  • DTL هو الأكثر أمانًا للمحتوى الديناميكي من المستخدمين.
  • Mako ممتاز للتطبيقات عالية الأداء، لكنه يتطلب رعاية دقيقة.
